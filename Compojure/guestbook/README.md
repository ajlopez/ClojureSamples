## Prerequisites

You will need [Leiningen][1] 1.7.0 or above installed.

[1]: https://github.com/technomancy/leiningen

## Running

To start a web server for the application, run:

```
lein ring server
```

## Using REPL

To start a web server for the application using the REPL, run:

```
lein repl
```

Then run
```
(use 'guestbook.repl)
(start-server)
```
